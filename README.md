# Weather-Forecast-System
This weather forecast is an web application that is built using OpenWeather API. This shows real time weather conditions like temp, humidity, wind speed.
PROBLEM STATEMENT:
Weather is a factor that keeps changing time to time, which affects our plan like: travelling, trekking, sports, farming.
This weather forecast project gives an easy & good solution to all the above mentioned problem.

OBJECTIVES OF PROJECT:
•	Giving appropriate information regarding weather of the entered city.
•	To program an app based on weather forecasting system.
•	Improving user interface with simple & accurate display.
•	To get data globally through API.

TECHNOLOGIES USED:
COMPONENTS	TECHNOLOGY
Backend	Python ,Flask
Frontend	HTML ,CSS 
Data Source	OpenWeatherMap API
Tools	Google Collab ,GitHub 
 
FUNCTIONALITIES:
FEATURES	DESCIPTION
Search City name	User input city name.
Live ,accurate data	App gives weather details with the help of API.
Display	Shows temperature & other weather details.
Error Management	It shows a message when invalid city name is entered.
